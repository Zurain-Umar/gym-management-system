# Gym Management System Website

## About This Project
This project is a simple Gym Management System website created using
Flask, HTML, and CSS. The website is designed for learning purposes
and helps beginners understand how a basic Flask web application works.

The project contains multiple web pages that are connected using
Flask routes and styled using CSS.

## Purpose of the Project
The main purpose of this project is to:
- Learn how Flask handles different web pages
- Understand how HTML templates work in Flask
- Learn how to use CSS for styling
- Practice creating a structured web project

## Pages in the Website

## How to Run This Project on Your Computer

1. Make sure Python is installed on your system.
2. Open the project folder in VS Code.
3. Open the terminal and install Flask using: pip install flask
4. Run the application by typing: python app.py
5. Open a web browser and go to: http://127.0.0.1:5000/


## Conclusion
This project is a beginner-friendly Flask web application that demonstrates
basic routing, template rendering, and website styling. It is suitable
for students who are learning web development using Flask.
