# Gym Management System Website

## About This Project
This project is a simple Gym Management System website created using
Flask, HTML, and CSS. The website is designed for learning purposes
and helps beginners understand how a basic Flask web application works.

The project contains multiple web pages that are connected using
Flask routes and styled using CSS.

## Purpose of the Project
The main purpose of this project is to:
- Learn how Flask handles different web pages
- Understand how HTML templates work in Flask
- Learn how to use CSS for styling
- Practice creating a structured web project
